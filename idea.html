Interactive Gulintangan

A simple browser-based musical learning app

💡 Project Idea

This app lets users play a two-row Gulintangan / Caklempong-style instrument directly in the browser. It is designed for learning, experimentation, and quick sharing through GitHub Pages.

Goal: Make traditional instrument exploration accessible with touch, mouse, and keyboard controls.

🎵 How It Works

Each circular pad triggers a synthesized bronze gong-like note using the browser Web Audio API.

  • Tap or click any pad to play.
  • Use keyboard keys A W S E D F T G Y H U J K for fast playing.
  • Multiple notes can overlap for chord-like or ensemble effects.

🌐 Technology Stack

  • HTML + CSS — static layout
  • JavaScript Web Audio API — generated sound
  • GitHub Pages — static hosting
