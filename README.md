# Interactive Gulintangan / Caklempong Audio Version

Open `index.html` in a browser, or upload all files to GitHub Pages.

Frontend layout was not changed. Only `main.js` was changed to load audio files from `assets/audio/`.

The included `.wav` files are generated caklempong-style placeholder samples so the project works immediately. For true original caklempong sound, record/replace these files while keeping the same filenames:

- C4.wav
- Cs4.wav
- D4.wav
- Ds4.wav
- E4.wav
- F4.wav
- Fs4.wav
- G4.wav
- Gs4.wav
- A4.wav
- As4.wav
- B4.wav
- C5.wav
