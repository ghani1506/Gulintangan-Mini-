# Interactive Gulintangan / Caklempong

A simple static web app that plays synthesized Gulintangan / Caklempong-style notes in the browser.

## Files

- `index.html` — main app page
- `main.js` — sound engine and controls
- `idea.html` — project concept page

## How to run locally

Open `index.html` in a modern browser, then click or tap any pad.

## How to launch on GitHub Pages

1. Upload these files to a GitHub repository.
2. Make sure the main page is named exactly `index.html`.
3. Go to **Settings → Pages**.
4. Select your branch, usually `main`, and root folder `/`.
5. Save and open the GitHub Pages URL.

## Controls

- Mouse/touch: tap pads
- Keyboard: `A W S E D F T G Y H U J K`

## Notes

This fixed version does not require external `.wav` files. Sounds are generated with the Web Audio API, so it works immediately on GitHub Pages.
